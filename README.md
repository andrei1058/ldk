# Information
* A multi-platform lgou2w development kit.

# Modules

* ldk
    * ldk-asm
    * ldk-common
    * ldk-i18n
    * ldk-reflect
    * ldk-retrofit
    * ldk-rx
    * ldk-fx
    * ldk-chat
    * ldk-nbt
    * ldk-bukkit
        * ldk-bukkit-common
        * ldk-bukkit-i18n
        * ldk-bukkit-scheduler
        * ldk-bukkit-plugin

# Development

* Intellij IDEA
* JDK 8
* Kotlin 1.2+
* Maven 3.3+
* Git

```bash
git clone https://github.com/lgou2w/ldk.git
cd ldk
mvn clean install
```
